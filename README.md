## Description

Raspberry Pi Zero 2W with the PiSugar Whisplay hat and PiSugar 3 battery to run a Chatbot service. 
The current implementation uses Whisper.cpp for TTS bash command execution and a SoftAP service for 
convenient ssh remoting to test, update, and debugging. 

### Goals
- Use PicoLM to run a small language model for executing bash commands based on spoken requests.
